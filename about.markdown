I will list my achievements on this project.
* Improving assembler API. 
* Support annotation tags
* Support java polymorphic calls in the interpreter
* Support ecmascript in ark interpreter
* Support AOT for ecmascript in ark compiler
* Improve perfomance for calls for ecmascript in compiler
